Db-profiler helps you understand how your database is used by WordPress and plugins

== Description ==

WordPress and plugins can have a significant impact on your database. When database tables expand in size performance drops. Sometimes a subtle option in a plugin makes a record to be written at every user access. This can crash your site if you have a decent amount of traffic.

== Installation ==

Installing "db-profiler" can be done either by searching for "db-profiler" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
